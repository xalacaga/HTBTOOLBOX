          Sous-dossier :
          
          

        

        
Renseigne ta clé API Claude (console.anthropic.com → API Keys) et clique ✦ Analyser le loot pour obtenir une analyse intelligente de ton audit.

L'IA lit tous les fichiers du dossier loot du domaine actif, identifie les vecteurs d'attaque (AS-REP, Kerberoast, ADCS ESC, SMB Relay, ACLs…) et propose un chemin d'exploitation concret.

La clé API est sauvegardée dans config.local.json (local, sur ta machine) via le bouton 💾 de la barre du haut.
